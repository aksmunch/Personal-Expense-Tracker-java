# Personal-Expense-Tracker-java
A simple and efficient Java-based application designed to help users record, manage, and analyze their daily expenses.
This project demonstrates core Java concepts including OOP, file handling, collections, recursion, custom exceptions, and modular programming.


------*Features*-----
-->Add new expenses (amount, category, date)
-->View all recorded expenses
-->Search expenses by category 
-->Sort expenses by amount 
-->total expenses


------*Data Storage*-------
-->Persistent storage using file handling (expenses saved automatically)
-->Reads & loads data at program startup

------*Java Concepts Implemented*------
Classes, Objects, Constructors
Inheritance, Abstract Classes, Interfaces
Polymorphism (Overloading & Overriding)
Custom Exceptions (Invalid Data)
Collections (ArrayList<Expense>)
File I/O (BufferedReader, BufferedWriter)
Recursion, Searching, Sorting


------*Project Structure*-------
PersonalExpenseTracker/
│── Main.java
│── Expense.java
│── ExpenseBase.java
│── FileOperations.java
│── ExpenseFileManager.java
│── InvalidDataException.java
│── expenses.txt        ← Auto-generated file for storing expenses
└── README.md
